O que é React?

React é uma biblioteca JavaScript para construir interfaces de usuário, desenvolvida e mantida pelo Facebook. Ele permite criar aplicações web interativas de forma eficiente e modular.

1. Componentes

Os componentes são a base do React. Eles permitem dividir a interface em partes reutilizáveis e independentes. Cada componente pode gerenciar seu próprio estado e lógica. Exemplo:

function MeuComponente() {
    return <div>Olá, Mundo!</div>;
}

2. Estado e Propriedades

O estado (state) é um objeto que armazena dados específicos do componente e pode mudar ao longo do tempo. As propriedades (props) são passadas para os componentes e são imutáveis. Exemplo:

function Contador(props) {
    const [contador, setContador] = React.useState(0);

    return (
        <div>
            <p>Contador: {contador}</p>
            <button onClick={() => setContador(contador + 1)}>Incrementar</button>
        </div>
    );
}

3. Ciclo de Vida

O ciclo de vida de um componente é o conjunto de métodos que são chamados em diferentes fases da existência do componente. Estes métodos permitem executar código em momentos específicos, como quando o componente é montado ou atualizado. Exemplo:

React.useEffect(() => {
    // Código a ser executado após a montagem do componente
    console.log('Componente montado');

    return () => {
        // Código a ser executado antes da desmontagem do componente
        console.log('Componente desmontado');
    };
}, []);
