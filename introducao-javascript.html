Introdução ao JavaScript

JavaScript é uma linguagem de programação essencial para o desenvolvimento web. Ela permite criar interatividade e dinamicidade em páginas da web. Vamos explorar alguns conceitos básicos:

1. Variáveis

Em JavaScript, variáveis são usadas para armazenar dados. Você pode declarar uma variável usando let, const ou var. Exemplo:

let nome = "João";

2. Funções

Funções são blocos de código projetados para realizar uma tarefa específica. Você pode definir uma função usando a palavra-chave function. Exemplo:

function saudacao() {
    console.log("Olá, Mundo!");
}

3. Eventos

Eventos são ações que ocorrem no navegador, como cliques e teclas pressionadas. Você pode adicionar eventos a elementos HTML usando JavaScript. Exemplo:

document.getElementById("meuBotao").addEventListener("click", function() {
    alert("Botão clicado!");
});
